## What It Is

OpenRouter provides access to many LLMs and providers through a single API layer. It is useful for comparing models, routing around outages, and managing cost/performance across providers.

## When To Use It

Use it when building AI apps or agents that need model flexibility, fallback options, or a faster way to test multiple providers without rewriting integrations.
